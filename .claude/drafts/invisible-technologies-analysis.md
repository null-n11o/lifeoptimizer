# AI企業 Invisible Technologies 徹底分析 | 評価額20億ドルのAIインフラ企業

## 導入部分（問題提起・共感）

2025年、生成AIブームが一段落し、本格的にAIを業務に組み込む「AIオペレーショナライゼーション」の時代に突入しています。

AIモデルのトレーニングやデータ準備を支援するScale AIが注目を集める中、その強力な競合として急成長しているのが今回紹介する**Invisible Technologies**です。

2025年9月に1億ドルの資金調達を行い、評価額20億ドル超のユニコーン企業となったInvisible Technologies。「AIと人間のハイブリッドアプローチ」という独自のビジネスモデルで、Microsoft、NBAのCharlotte Hornetsなどのエンタープライズ顧客を獲得しています。

今回はこのInvisible Technologiesについて徹底的に分析していきます。

2023年にPedrazaはCEO職を退き、Executive Chairman（取締役会長）に就任。その後も企業の戦略的方向性を導いています。

## ビジネスモデル: AIと人間のハイブリッドアプローチ

Invisible Technologiesの最大の特徴は、**AIプラットフォームと人間の専門知識を組み合わせたハイブリッドアプローチ**です。

完全自動化ではなく、AIと人間のエキスパートが協働することで、より高品質なAIモデルのトレーニングとデータ処理を実現しています。

### なぜハイブリッドなのか: AIプロジェクトの95%失敗問題

多くの企業がAI導入で失敗する理由は、**「ハードヤード（困難な準備作業）」を省略するから**です。

システムを整備せずにいきなりAIを導入しようとし、事前の人的作業を避けようとする。その結果、AIプロジェクトの**95%が失敗**しているのが現実です。

Invisibleのアプローチは真逆です。エンジニアを現場に配置し、実際の業務フローを徹底的にマッピングし、耐久性、適応性、精度を備えたシステムを構築してからAIを投入します。この「ハードヤード」を避けない姿勢が、Invisibleの高い成功率の秘訣ですね。

### 二つの主要事業

Invisibleは大きく分けて二つの事業を展開しています。

#### **A. AIトレーニング事業**

- 世界トップAIモデルの80%以上のトレーニングに貢献
- **OpenAIのGPT-3.5開発**に協力（ChatGPT初期バージョン）
- **Cohere**のエージェントタスク性能向上（競合他社を上回る結果）
- **Microsoft**、**Amazon AWS**、**AI21 Labs**との戦略的パートナーシップ
- データクリーニング、ラベリング、構造化サービス
- 専門分野エキスパートによる高品質アノテーション

#### **B. エンタープライズAIソリューション**

企業向けに包括的なAI導入支援を提供。2024年の売上1.34億ドルのうち「相当な割合」がこの事業から生まれています。

### 5つのモジュラーコンポーネント

Invisibleのプラットフォームは以下の5つのコンポーネントで構成されています。

1. **データ構造化モジュール**
   - 未整理データのクリーニングとラベリング
   - AI活用可能な形式への変換
   - 混沌としたデータを構造化

2. **ワークフロービルダー**
   - デジタルワークフローの構築
   - 業務プロセスのAI対応化
   - プロセスを管理可能なブロックに分割

3. **カスタムAIモデル構築**
   - 企業固有データで微調整されたモデル開発
   - AIエージェント型ソリューションの展開
   - 自律的タスク実行の実現

4. **評価・測定フレームワーク**
   - AIモデルの性能測定とテスト
   - パフォーマンス評価
   - ROI測定

5. **エキスパートマーケットプレイス**
   - 関連する人間専門家の動員
   - 医療、法律、金融など特定分野の深い知識を持つエキスパート
   - ハイブリッドアプローチの実現

これらのツールを組み合わせることで、企業は自社のAI開発ニーズに応じて柔軟にシステムを構築できるわけです。

### グローバルエキスパートネットワーク

Invisibleの強みは、**70カ国以上**にまたがる**グローバル専門家ネットワーク**です。

AIモデルのトレーニングには「正解」を定義する必要がありますが、その作業を特定分野の専門知識を持つ人々に一時的に依頼し、高品質なデータアノテーション（データラベリング）やモデル評価を行う仕組みを構築しています。

興味深いのは、Invisibleのシステムを形作っているのが**「世界トップAIモデルの80%をトレーニングした人材」**だということ。まさにAI業界の最前線にいる人たちがこのプラットフォームを支えているわけですね。

ハイブリッドリモート体制で、サンフランシスコ、ニューヨーク、ロンドンのコラボレーションハブを拠点に、グローバルに分散したチームが協働しています。

### 強化学習ジム（RL Gyms）: AIエージェントの訓練場

Invisibleが開発している革新的な技術の一つが**強化学習ジム（Reinforcement Learning Gyms）**です。

これはAIエージェントが仮想環境で学習できる訓練場で、例えば**Microsoft Teams**のような実際のアプリケーションをシミュレーションし、AIエージェントが実環境に投入される前に戦略を習得できる仕組みです。

この技術により、AIエージェントは安全な環境で試行錯誤を繰り返し、実際の業務環境に投入される時点で既に高い精度を持つことができます。まさにAIの「訓練場」ですね。

創業者のFrancis Pedrazaという人物も興味深いです。

単にビジネスで成功するだけでなく、Infinity Constellation（AIサービス業界のバークシャー・ハサウェイ）やEternity Foundation（600万ページの古代テキストのAI翻訳）といったプロジェクトを通じて、AI業界全体の発展や文化的貢献にも取り組んでいる姿勢は、長期的な視点を持った起業家の姿勢だと感じます。

### 今後の展望

2025年に新CEOのMatt Fitzpatrick（元McKinsey QuantumBlack Labsヘッド）が就任し、組織が急拡大している中で、今後Invisibleがどのように進化していくのか注目していきたいと思います。

特に、強化学習ジム（RL Gyms）のような革新的な技術や、エージェント型AIワークフローの拡大など、次世代のAIインフラを構築しようとしている点が楽しみです。

Scale AIとの競争も激しくなるでしょうが、ハイブリッドアプローチという差別化要因を活かして、さらなる成長を遂げるのではないでしょうか。

## まとめ: Invisible Technologiesの重要ポイント

### 企業概要
- **評価額20億ドル超のAIインフラ企業**（2025年9月、1億ドル調達）
- **2015年創業**、10年で約3,000名超の組織に成長

- **AIと人間のハイブリッドアプローチ**で高品質なAIシステムを構築
- **「ハードヤード」を避けない姿勢**: 事前準備を徹底し、AIプロジェクトの95%失敗問題を克服
- **5つのモジュラーコンポーネント**でエンタープライズAIをオペレーショナライズ
- **70カ国以上のグローバルエキスパートネットワーク**
- **強化学習ジム（RL Gyms）**: AIエージェントの訓練環境を提供

### 成長実績
- **4年間で48倍成長**: 2020年300万ドル → 2024年1.34億ドル

- **顧客ROI平均9〜10倍**: 具体的な成果で価値を証明

- **OpenAI（GPT-3.5開発協力）**、**Cohere**、**Microsoft**、**Amazon AWS**
- **Nasdaq、Headway、Big 4小売業者、Charlotte Hornets**など多様な業界で成功事例
- コスト削減90%、処理速度8倍向上、ROI 9倍などの具体的成果
